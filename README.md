# 🕐 Digital Clock

A simple real-time digital clock built with Python.

## Features
- Shows time in 12-hour format with AM/PM
- Updates every second
- Clean terminal display
- Only 6 lines of code!

## How to Run
```bash
python digital_clock.py
# Press Ctrl+C to stop
```

# Requirements
· Python 3.x
· No external libraries needed!
